Simple Calculator

A beginner-friendly calculator built using HTML, CSS, and JavaScript.
It performs basic arithmetic operations and is perfect as a first JavaScript project.


Features
	•	Addition, subtraction, multiplication, division
	•	Supports decimal numbers
	•	Clear display button
	•	Clean and simple UI
	•	Uses vanilla JavaScript (no frameworks)

  
Project Files
  index.html  
  style.css  
  script.js  

  
How to Use
	1.	Enter numbers by clicking the buttons.
	2.	Click operators like +, -, *, /.
	3.	Press = to see the result.
	4.	Press Clear Display to reset the screen.


Code Overview
	•	appendToDisplay(value) → Adds numbers/operators to the screen
	•	calculateResult() → Uses eval() to calculate the expression
	•	clearDisplay() → Empties the input field


How to Run

Just open index.html in any browser. No setup required.


Technologies Used
	•	HTML
